How to Use LYTE Blog Embed

1. Add the Script

<script src="https://your-domain.com/path/to/blog-embed.js" type="module"></script>

2. Add the Component

<lyte-blog 
    data-token="YOUR_EMBED_TOKEN" 
    theme="light|dark" 
    posts-per-page="10">
    <noscript>
        Please enable JavaScript to view our blog posts.
    </noscript>
</lyte-blog>

Attributes

  • data-token Your unique embed token (required). Get this from your LYTE dashboard.
  • theme "light" or "dark" (optional, defaults to "light"). Controls the color scheme of the blog posts.
  • posts-per-page Number of posts to display (optional, defaults to 10). Maximum value is 50.

Security Notes

  • Keep your embed token secure. It's tied to your subscription.
  • The embed token will stop working if your subscription becomes inactive.
  • You can regenerate your embed token from the LYTE dashboard if needed.

Customization

The blog embed automatically adapts to your website's font settings and provides a responsive layout. For advanced customization, contact our support team.
