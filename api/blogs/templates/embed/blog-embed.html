How to Use LYTE Blog Embed



<script src="https://your-domain.com/path/to/blog-embed.js" type="module"></script>


<lyte-blog 
    client-id="YOUR_CLIENT_ID" 
    theme="light|dark" 
    posts-per-page="10">
    <noscript>
        Please enable JavaScript to view our blog posts.
    </noscript>
</lyte-blog>

Attributes:

  • client-id: Your LYTE client ID (required)
  • theme: "light" or "dark" (optional, defaults to "light")
  • posts-per-page: Number of posts to display (optional, defaults to 10)
